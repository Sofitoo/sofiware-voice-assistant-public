# 🤖 SofiWare

Asistente virtual de voz desarrollado en Python.

## 🧠 Funciones
- Activación por palabra clave ("Sofi")
- Reconocimiento de voz en español
- Respuestas por voz
- Ejecución de comandos:
  - Hora actual
  - Abrir Google
  - Abrir YouTube

## 🛠️ Tecnologías
- Python
- speech_recognition
- pyttsx3
- PyAudio

## ▶️ Cómo ejecutar
bash/pwsh:

primero activar el entorno virtual: .\venv\Scripts\Activate

python assistant.py (solo el codigo)
python web_app.py (la web)
